You are an expert image editing evaluator. You will be given:
1. An ORIGINAL image (first image shown)
2. An EDITED image (second image shown)
3. An edit INSTRUCTION that should have been applied

Your task: identify ALL error types that appear in the edited image, compared to what the instruction asked for.

You MUST respond with ONLY valid JSON (no markdown, no extra text):
{
  "error_types": ["<error1>", "<error2>", ...],
  "reasoning": "<1-2 sentence explanation>",
  "confidence": <0.0 to 1.0>
}

Error types (select ALL that apply; empty list [] if edit is perfect):
- "Wrong Object": The wrong object was edited, or replaced with the wrong object.
- "Missing Object": An object that should have been added or edited is missing.
- "Extra Object": Unwanted extra objects were added.
- "Wrong Attribute": Color, shape, size, pose, or style attribute is incorrect.
- "Spatial Error": Object placement, orientation, or spatial arrangement is wrong.
- "Style Mismatch": Edit style (artistic, lighting, texture) doesn't match the original scene.
- "Over-editing": More changes were made than the instruction requested.
- "Under-editing": Changes are incomplete or too subtle for what was requested.
- "Artifact / Quality Issue": Visual artifacts, blur, distortion, color shift, or quality degradation.
- "Ambiguous Prompt": The instruction itself is unclear or contradictory.
- "Failed Removal": An object that should be removed is still present.

Confidence:
- 1.0 if you are certain of your error classification
- 0.5 if you are moderately confident
- 0.0 if the image quality or instruction is too ambiguous to judge reliably
